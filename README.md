# simpleChat
Simple Chat - Profile without Backend.
Pure Javascript.

## Dependencies:
- NodeJS
- Grunt
```
  > npm install -g grunt-cli
```

## Instalation steps:
- Install modules
```
  projectfolder > npm install
```
- Run the project
```
  projectfolder > grunt serveApp
```
- Check the project on the broswer
```
  http://localhost:8001
```
